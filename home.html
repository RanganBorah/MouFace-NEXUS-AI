Project Highlights

📸 Face Registration

Captures 50 processed face samples per user and prepares clean data for model training.

🎯 Real-Time Recognition

Runs continuous face recognition using a live browser camera feed and backend prediction.

🤖 AI Pipeline

Uses Haar Cascade detection, grayscale normalization, histogram equalization, and LBPH recognition.

💾 Database Integration

Stores user details in MySQL and fetches profile information during recognition.
